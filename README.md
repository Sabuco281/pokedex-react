# Pokedex en React

Este proyecto es una aplicacion web sencilla creada con React y Vite para practicar componentes, renderizado de listas y organizacion basica de datos.

## Que hace

- Muestra una lista de Pokemon en tarjetas.
- Cada tarjeta presenta nombre, tipo, descripcion y generacion.
- Usa un dataset local (`src/data/pokemons.js`) para renderizar el contenido.

## Tecnologias

- React 18
- Vite 5
- CSS puro

## Scripts disponibles

- `npm run dev`: inicia el entorno de desarrollo.
- `npm run build`: genera la version de produccion.
- `npm run preview`: previsualiza el build de produccion.

## Estructura principal

- `src/App.jsx`: vista principal y render de la lista.
- `src/components/PokemonCard.jsx`: componente de tarjeta.
- `src/data/pokemons.js`: datos base de Pokemon.
- `src/styles.css`: estilos globales.
